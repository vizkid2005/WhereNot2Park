#WHERE NOT TO PARK
===================================

Ever got your vehicle towed just because you didn't get a place to park in a hurry ? Always wished to get assistance about where to park ?
This app is exactly the assistance yourequire. All you gotta do is login to the app using Google and mark nearby places where parking is available. This will help you and your co-app users to know WHERE TO PARK the next time they are there!


Pre-requisites
--------------
Android SDK 21 or Higher
Build Tools version 21.1.2
Android Support AppCompat 22.2.0
Google Play Services Map 7.0.0

Getting Started
---------------
This sample uses the Gradle build system.  To build this project, use the
"gradlew build" command or use "Import Project" in Android Studio.
